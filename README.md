# Water Reminder Assistant

A simple desktop water reminder application built with Python and PySide6.

## Features

- 💧 Periodic water reminders
- 🖼️ Animated desktop companion
- 💬 Interactive reminder popup
- 🖥️ Transparent desktop overlay

## Technologies Used

- Python
- PySide6
- PyInstaller
